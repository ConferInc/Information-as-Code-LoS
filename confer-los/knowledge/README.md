# Knowledge Base

This directory contains the immutable source of truth for Confer LOS.

## Structure

- **[systems/](systems/)**: Technical stack, configurations, architecture.
    - `01-architecture-map.md`: System components and dependency graphs.
    - `02-data-flow.md`: Detailed user journeys and logic flows.
    - `03-infrastructure.md`: Database schema, Docker setup, and CI/CD.
- **[processes/](processes/)**: Business workflows, decision trees.
- **[people/](people/)**: Team directory (org chart, domains).

## Guidelines
- Changes here affect the core "truth" of the project.
- Major changes should be accompanied by a Decision Record.
- Consult this folder *before* starting new work.
