# Meetings

This directory contains meeting notes organized by **Year-Month**.

## Structure
```
meetings/
├── README.md
├── 2025-01/
│   ├── 2025-01-15-standup.md
│   └── 2025-01-16-planning.md
└── 2025-02/
```

## Workflow
1. Create a directory for the current month if it doesn't exist (e.g., `2025-03`).
2. Create a new file using the `meeting.md` template.
3. Filename format: `YYYY-MM-DD-topic.md`.
